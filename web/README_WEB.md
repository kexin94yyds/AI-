# AI 全家桶 - 网页版 (Web Version)

这是一个纯网页版本的 AI 聚合应用，专为移动端和普通浏览器设计。

## 特性
- 📱 **移动端优先**：在手机上自动切换为底部导航栏，体验接近原生 App。
- 🖥️ **桌面端适配**：电脑上保持左侧边栏布局。
- 🔗 **链接聚合**：一键直达所有主流 AI 服务。

## 部署与使用

### 本地使用
1. 在项目根目录下启动一个静态服务器（例如使用 Python）：
   ```bash
   python3 -m http.server 8000
   ```
2. 浏览器访问：`http://localhost:8000/web/index.html`
3. 手机访问：确保手机和电脑在同一 WiFi，使用电脑 IP 访问，如 `http://192.168.1.x:8000/web/index.html`。

### 部署到公网 (Vercel/Netlify/GitHub Pages)
1. 将整个项目上传到 GitHub。
2. 在 Vercel 等平台导入项目。
3. 将 **Output Directory** (发布目录) 设置为 `web` 或者保持默认，但需要通过 `/web/index.html` 访问。

## 注意事项 (Web 版 vs 插件版)
由于浏览器的安全限制 (Same-Origin Policy)，网页版与插件版有以下区别：

1. **无法直接嵌入**：大多数 AI 网站（如 ChatGPT, Claude）禁止在 iframe 中加载。因此网页版默认会显示"在新窗口打开"的提示，这在手机上体验更好（全屏使用）。
2. **历史记录**：由于无法监控你在第三方网站内的跳转，网页版**无法自动记录**具体的对话历史链接，只能记录你最后点击了哪个提供商。
3. **跨域限制**：如果想在网页内直接嵌入，需要安装允许跨域的浏览器插件（如 "Allow CORS"），但这不推荐在手机上折腾。
